Welcome to the Bookmarks Web Service, a bookmarking and and link shortening web service with REST API endpoints.

With this web service, users can save URLs to bookmark IDs (unique, random 6 character alphanumeric sequences). Users can access the saved URL by making a request to the saved bookmark\'s endpoint.

To create an account, please email the administrator: {{ config['ADMIN_EMAIL'] }}.

Once you receive your User ID and password, you will be able to create an API Key, which you can use to create and access bookmarks.

Please refer to the API documentation.
